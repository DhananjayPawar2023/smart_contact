====================================================================================
⚠️ TROUBLESHOOTING:
====================================================================================

PROBLEM: "ethers.js not loaded"
SOLUTION: Check HTML script tag: <script src="https://cdnjs.cloudflare.com/ajax/libs/ethers.js/6.4.0/ethers.umd.min.js"></script>

PROBLEM: "Contract not loaded"
SOLUTION: Verify CONTRACT_ADDRESS in app.js (should start with 0x)
          Verify CONTRACT_ABI is valid JSON from Remix

PROBLEM: "Can't connect to MetaMask"
SOLUTION: Install MetaMask extension in browser
          Create wallet if not already created
          Add test ETH from faucet
          Select MetaMask when browser asks

PROBLEM: "Transaction failed"
SOLUTION: Check network is Sepolia testnet
          Check you have test ETH in wallet
          Check gas price isn't too high

PROBLEM: "Already rated error" ✅
SOLUTION: This is CORRECT! Feature works!
          Your contract prevents duplicate ratings (intentional)
