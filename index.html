Simple College Project

Movie Rating Using Remix VM

No MetaMask flow. Deploy the contract in Remix VM, open this UI as a Remix plugin, select a funded Remix account, and rate movies directly from the frontend.

Simple Flow

1. Run npm start or node server.js

2. Open this page from Remix Plugin Manager

3. Deploy contracts/MovieRating.sol in Remix VM (London)

4. Paste contract address and rate from the UI

Contract

The contract stores only ratings on-chain using each movie's IMDb ID. The movie list itself stays local and simple.

No Wallet Setup

The UI uses Remix's own funded VM accounts through the Remix plugin bridge. Gas comes from Remix VM, not from MetaMask.

Why This Version

The old project mixed MetaMask docs with a fake demo UI. This version keeps one real workflow that is easier to present and easier to run.

How To Run

1. Start this project with npm start or node server.js. 2. In Remix IDE, open Plugin Manager. 3. Use Connect to a Local Plugin and enter this app URL. 4. In Remix, compile and deploy contracts/MovieRating.sol using Remix VM (London). 5. Copy the deployed contract address. 6. In this UI, click Connect to Remix. 7. Paste the contract address and click Refresh Ratings. 8. Pick any Remix account and click a rating button.
